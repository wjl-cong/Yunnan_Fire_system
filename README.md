#### 软件架构

**技术栈：**

- **前端框架**：Vue3（响应式比云南的天气变化还快）
- **地图引擎**：OpenLayers（地图界的"老司机"）
- **图表库**：ECharts（数据界的"画家"）
- **UI组件**：ElementPlus（颜值担当）
- **数据格式**：GeoJSON（地理数据的"通用语言"）

**系统架构：**
- 前端：Vue3单页面应用(SPA)
- 地图服务：OpenLayers + 高德地图API
- 数据可视化：ECharts图表库
- 数据存储：本地GeoJSON文件 + 在线API

#### 使用说明

**"傻瓜式"操作指南：**

1. **查看火险等级分布**
   - 打开地图，五颜六色的区域就是火险等级
   - 蓝色=较低（可以"散步"）
   - 黄色=中等（需要"注意"）
   - 红色=极高（需要"紧张"）
2. **使用图层控制**
   - 点击图层管理按钮，像"开关灯"一样控制图层显示
   - 可以叠加多个矢量图等
3. **测量工具使用**
   - 测距：点击"测量距离"图标，丈量长度
   - 测面：点击"测量面积"图标，计算面积
4. **热力图分析**
   - 红色越亮，火险越高（"热点"地区）
   - 可以用来找"最危险"的区域
5. **详情查询**
   - 点击地图上的点位，弹出"八卦"窗口
   - 查看详细火险信息、历史数据
